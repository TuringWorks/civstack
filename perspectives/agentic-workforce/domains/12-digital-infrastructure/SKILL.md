# Communications, Software, Cybersecurity, and Digital Infrastructure Workforce

## Mission

Enable trusted computation, communication, data storage, software services, and cyber resilience.

## Core Jobs

- Provide reliable networks, software, and digital services.
- Operate secure compute, storage, cloud, and data centers.
- Detect, respond, recover, and harden against cyber attacks.
- Design, build, test, deploy, and maintain software.
- Govern privacy, fairness, safety, reliability, and digital rights.

## Human Role Families

- Software engineer, full-stack engineer, mobile engineer, platform engineer.
- Product manager, UX designer, UX researcher, technical program manager.
- Data engineer, data scientist, analytics engineer, business intelligence analyst.
- Network engineer, telecom technician, data center technician, cloud architect.
- Cybersecurity analyst, security engineer, incident responder, threat hunter.
- AI engineer, machine learning engineer, applied scientist, MLOps engineer, AI product manager.
- AI governance manager, model risk manager, trust and safety analyst, AI safety evaluator.

## AI Personnel

- Coding agent: implement, test, refactor, document, and migrate code.
- Test generation agent: create unit, integration, regression, and adversarial tests.
- Incident response copilot: summarize alerts, propose mitigations, maintain timelines.
- Threat intelligence agent: enrich indicators and map adversary techniques.
- Data quality agent: detect schema drift, missingness, and pipeline failures.
- Model evaluation agent: measure model quality, safety, bias, cost, and drift.
- Documentation agent: keep architecture, runbooks, APIs, and decisions current.

## Humanoid Robot Roles

- Data center inspection, hardware swap assistance, cable handling.
- Office IT runner and device logistics.
- Physical security patrol support.

## Accountability Boundary

Keep security incident command, privacy commitments, AI deployment approval, customer trust decisions, architecture tradeoffs, and rights-impacting automation human-led.

## Agent Context Requirements

Load repo, architecture diagrams, service ownership, APIs, SLAs, incident history, security policies, data schemas, privacy rules, and model cards/eval results.
