# Lamina Concept Image Archive

These images document the design progression. They are concept studies rather than validated mechanical drawings.

| File | Design stage | Purpose |
|---|---|---|
| `lamina-fixed-base-v0.png` | Fixed-base upper-body demonstrator | Establishes the layered-plywood visual language, exposed joint cartridges, and weighted development pedestal. |
| `lamina-mobile-worker-v1.png` | Wheeled task robot | Explores two-arm manipulation, torso lift, adaptive grippers, and low-risk indoor mobility. |
| `lamina-wooden-biped-map-v2.png` | Wooden biped architecture map | Maps plywood torsion boxes, tendon actuation, modular interfaces, stairs, squatting, grip, and tethered ladder testing. |
| `lamina-slender-biped-v3.png` | Slender strength-preserving biped | Refines the biped with proximal motors, tapered hollow limbs, counterbalance, structural carrying surfaces, and positive ladder grip. |
| `lamina-slender-biped-v3-bom.svg` | Visual bill of materials | Editable exploded component map for the slender biped; exact quantities live in `../hardware/slender-biped-v3-bom.csv`. |
| `lamina-slender-biped-v4-rear-estop.png` | Rear safety-control refinement | Shows the fixed shoulder-height rear E-stop, separate overhead tether anchor, slender wooden structure, and light-load carrying pose. |
| `lamina-freecad-v4-preview.png` | FreeCAD visibility smoke test | Rendered directly from the native v4 `.FCStd` after GUI view providers and visibility metadata are saved. |
| `lamina-freecad-v5-front.png` | Image-aligned FreeCAD v5 front | Front visibility render of the open-truss v5 architecture model. |

## Design interpretation

The generated visuals can contain mechanically ambiguous details. Fabrication should follow the written architecture, measured load cases, joint-clearance studies, and safety tests—not inferred geometry from the images.
