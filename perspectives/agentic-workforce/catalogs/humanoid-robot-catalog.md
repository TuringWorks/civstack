# Humanoid Robot Catalog

Use humanoids where human-shaped mobility/manipulation is useful in existing spaces. Prefer specialized automation where the environment can be redesigned.

| Robot Role | Jobs To Be Done | Best Environments | Human Supervisor |
|---|---|---|---|
| Material runner | Move tools, supplies, linens, parts | Hospitals, hotels, plants, offices | Site operations lead |
| Inspection walker | Patrol, inspect, photograph, report | Utilities, plants, warehouses | Maintenance/safety manager |
| Warehouse associate | Pick, pack, sort, replenish | Warehouses, retail backrooms | Warehouse manager |
| Facilities maintainer | Clean, restock, check rooms | Schools, offices, stations | Facilities manager |
| Care support aide | Fetch, remind, lift-assist, monitor | Hospitals, eldercare, homes | Nurse/caregiver |
| Lab assistant | Move samples, load instruments | Labs, pharma, hospitals | Lab manager |
| Disaster support unit | Enter risky areas, carry supplies | Fires, floods, industrial accidents | Incident commander |
| Retail/hospitality helper | Retrieve items, deliver orders | Stores, hotels, restaurants | Store/service manager |
| Manufacturing cell worker | Tend machines, assemble, inspect | Brownfield factories | Production supervisor |
| Farm/greenhouse helper | Harvest, sort, pack, inspect | Greenhouses, controlled farms | Farm operations lead |

Common stop conditions: human distress, blocked path, uncertain object, access-control conflict, chemical/electrical/fire hazard, unexpected human contact, damaged goods, command conflict, sensor failure.
