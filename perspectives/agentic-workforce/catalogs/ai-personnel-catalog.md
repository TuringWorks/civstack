# AI Personnel Catalog

Use these AI-personnel patterns as reusable building blocks. Every AI role needs a human owner, explicit permissions, logging, evaluation, and escalation rules.

| AI Personnel | Jobs To Be Done | Strong Inputs | Outputs | Human Owner |
|---|---|---|---|---|
| Research analyst agent | Gather, compare, cite, summarize | Documents, web, databases | Briefs, evidence tables | Analyst/scientist/attorney |
| Policy simulator agent | Model policy options and impacts | Baselines, constraints, objectives | Scenarios, tradeoff maps | Policymaker |
| Operations coordinator agent | Route work and monitor queues | Tickets, telemetry, schedules | Assignments, alerts | Operations manager |
| Drafting agent | Produce first drafts | Templates, facts, policies | Memos, SOPs, contracts | Domain owner |
| Compliance review agent | Compare evidence to rules | Regulations, logs, files | Findings, audit trails | Compliance officer |
| Citizen/customer service agent | Intake and resolve routine requests | KB, account data, forms | Answers, cases, escalations | Service manager |
| Coding agent | Build/test/refactor software | Repo, issues, tests | Code, tests, docs | Engineer/tech lead |
| Cyber triage agent | Classify alerts and enrich incidents | SIEM, logs, threat intel | Incident notes, recommendations | SOC lead |
| Model evaluation agent | Test AI quality/safety/drift | Prompts, eval sets, traces | Scores, failures, risks | AI governance lead |
| Procurement agent | Compare suppliers and draft RFPs | Requirements, vendor data | RFPs, scorecards | Procurement officer |
| Finance operations agent | Reconcile and detect anomalies | Ledger, invoices, market data | Exceptions, forecasts | Controller/CFO |
| Field-service planner agent | Predict failures and schedule crews | Asset telemetry, inventory | Work orders, parts lists | Maintenance manager |
| Tutor/trainer agent | Adapt instruction and practice | Curriculum, learner data | Lessons, quizzes, feedback | Teacher/trainer |
| Personal admin agent | Coordinate everyday administration | Calendar, messages, forms | Plans, reminders, drafts | Individual/family |
