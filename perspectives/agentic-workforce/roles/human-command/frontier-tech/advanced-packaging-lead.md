# Advanced Packaging Lead

## Mission

Integrate chiplets, memory, substrates, thermal systems, and interconnects into high-performance packages for AI and strategic computing.

## When Needed

- Performance depends on chiplet integration, memory bandwidth, thermal management, or package-level yield.
- Domestic chip capability needs packaging capacity as much as wafer fabrication.
- Supply chain or export constraints affect substrates, interposers, and assembly.

## Core Jobs To Be Done

- When chiplets must become systems, design package architecture so performance, thermals, reliability, and manufacturability align.
- When package yield fails, diagnose assembly, substrate, thermal, and test issues.
- When demand scales, qualify suppliers and capacity.

## Responsibilities

- Own package roadmap, design rules, supplier qualification, assembly/test flow, thermal strategy, and reliability validation.

## AI Delegation

- Package simulation agent, thermal analysis agent, supplier risk agent, yield correlation agent, test coverage assistant.

## Robot Delegation

- Cleanroom/material logistics, inspection, test hardware movement.

## Human Boundary

Package architecture, supplier qualification, reliability release, customer commitments, and capital investment remain human-led.

## Metrics

- Package yield.
- Thermal performance.
- Reliability test pass rate.
- Assembly cycle time.
- Memory/interconnect performance.
- Supplier resilience.

## Context Needed By Agents

Chiplet specs, substrate constraints, thermal models, assembly process, test data, supplier data, reliability standards, and customer requirements.
