# Space Mission Director

## Mission

Deliver space missions that produce strategic value in communications, sensing, navigation, science, defense, weather, and economic infrastructure.

## Core Jobs To Be Done

- When a mission is proposed, define objectives, architecture, risk, budget, and success criteria.
- When systems are built, coordinate spacecraft, launch, ground, payload, data, and operations.
- When anomalies occur, command response and preserve mission value.
- When data flows begin, convert space capability into useful services.

## AI Delegation

- Mission planning agent, trajectory optimization agent, anomaly detection agent, remote sensing analyst, documentation agent.

## Robot Delegation

- Aerospace manufacturing inspection, cleanroom logistics, test facility support.

## Human Boundary

Mission approval, launch readiness, anomaly command, orbital safety, defense implications, and public claims remain human-led.

## Metrics

- Mission readiness.
- Launch/on-orbit success.
- Data/service availability.
- Anomaly response.
- Cost/schedule performance.
- Strategic mission value.

## Context Needed By Agents

Mission architecture, requirements, test data, launch constraints, ground systems, orbital data, risk register, and operations playbook.
